A modern, responsive personal website built with Next.js 14, TypeScript, and TailwindCSS.

## Getting Started

1. Install dependencies
```bash
npm install
```
2. Run the dev server
```bash
npm run dev
```
3. Open http://localhost:3000

## Customize Content
- Edit `data/profile.ts` to update your name, summary, projects, skills, education, certificates, and contact.
- Components live in `components/`.
- Styles are in `app/globals.css` and Tailwind config in `tailwind.config.ts`.

## Build
```bash
npm run build && npm start
```
